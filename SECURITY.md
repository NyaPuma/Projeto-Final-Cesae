# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x | ✅ |
| <1.0 | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not create a public issue.

Instead, contact:

- Email: security@example.com

Include:

- Description
- Steps to reproduce
- Impact
- Possible mitigation

A response will normally be provided within 5 working days.
